项目简介

Raoailongxia AI龙虾项目（v0.0.01版本）是面向龙虾养殖/分析场景的AI辅助系统，旨在通过人工智能技术解决龙虾养殖过程中的数据监测、病害识别、产量预测等核心问题，降低养殖成本，提升养殖效率。

当前v0.0.01版本为基础原型版本，聚焦核心功能的验证与落地，后续将持续迭代优化。

v0.0.01 核心功能

  • ✓ 龙虾养殖环境数据（水温/溶氧量）实时采集与可视化
  • ✓ 基础龙虾病害图像识别（支持3类常见病害初步判定）
  • ✓ 养殖数据简易统计报表生成
  • ✓ 本地数据存储与基础导出功能

版本说明

v0.0.01 为项目首个原型版本，完成核心技术栈的搭建与基础功能验证：

  • ✓ 前端：HTML5 + CSS3 + JavaScript 原生开发，适配移动端/PC端基础布局
  • ✓ 后端：暂未接入，v0.0.01版本为前端静态演示版
  • ✓ AI能力：集成轻量级图像识别模型（本地离线版）
  • ✓ 待优化：病害识别准确率、数据同步、多端适配细节
